e-Rapor V2.5

Aplikasi cetak rapor berbasis web — 100% offline, langsung pakai tanpa instalasi database. Cukup upload file Excel leger nilai, isi pengaturan, lalu cetak rapor per siswa atau per kelas.

1 Download sample Excel → isi nilai sesuai kolom
2 Upload file Excel leger nilai di bawah
3 Klik Buka Rapor → pilih kelas → pilih siswa
4 Atur identitas sekolah, TTD, dan wali kelas lewat tombol 🔧 Pengaturan
5 Cetak rapor dengan tombol 🖨 Cetak / PDF
